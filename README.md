# Airplane-Wing-Analysis-ANSYS

### About
Structural and CFD analysis of an 
airplane wing using ANSYS 2026 R1.

### Tools Used
- ANSYS Static Structural
- ANSYS Fluent (CFD)
- DesignModeler

### Structural Analysis Results
- Total Deformation: 3.48e-10 m (Max)
- Normal Stress: 109.4 Pa (Max)
- Strain Energy: 3.25e-11 J (Max)

### CFD Results
- Drag Coefficient (Cd): 0.0769
- Lift Coefficient (Cl): 1.9393
